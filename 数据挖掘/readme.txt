数据挖掘流程

1.数据读取
    a.读取数据,并进行显示
    b.统计数据各项指标
    c.明确数据规模与要完成的任务
2.特征理解分析
    a.单特征分析,逐个变量分析其对结果的影响
    b.多变量统计分析,综合考虑多种情况影响
    c.统计绘图,得出结论
3.数据清洗与预处理(比较难做,也是最重要的)
    a.对缺失值进行填充
    b.特征标准化/归一化
    c.筛选有价值的特征(奖金,工资)
    d.分析特征之间的相关性(希望特征之间存在差异性)
    note:数据清洗很重要,数据决定了模型的上限,算法只是怎么逼近这个上限

4.建立模型
    a.特征数据与标签准备
    b.数据集切分
    c.多种建模算法对比
    d.集成策略等方案改进
