1.两台电脑之间的通信
    两台电脑要通信必须<网络号>相同
    网络号 = ip&子网掩码
    ip = 网络号+主机号

    如一台机器,ip 192.168.1.1 子网掩码255.255.255.0
    则网络号:(192.168.1.1)&(255.255.255.0)=192.168.1.0
    主机号:ip-网络号= 0.0.0.1

2.集线器
  简称hub,现在已经不使用了
  多台电脑相连，以广播的形式发信息。其中一台电脑发信息时，其他电脑不可发。工作在物理层
3.交换机
  优化集线器，实现自动寻址，使得一台电脑可以和另外一台电脑通信而不采用广播的形式。
  一台电脑发信息时，其他电脑也可以发。工作在数据链路层
  可以广播也可以单播
    广播:向连接的各个电脑发送arp广播,寻找特定ip的mac地址
    单播:接收到arp广播的电脑,校验信息,发现是自己的ip,将自己的mac地址以单播的形式发给交换机

4.路由器
  至少有两个网卡，链接两个不同的网络

5.网关
  有数据转发能力的设备，将一个网络的数据转发到另外一个网络，一般由路由器担任
  当一台电脑发送数据的目标ip地址和本机地址不在一个网络时，就将数据发送至网关，由网关转发出去

一个数据包从一个网络转发到另外一个网络时，数据包ip地址是不变的，但是数据包的mac会变
(每经过一个设备如路由器转发，目标和源ip不变，但目标和源mac变)

同一个网络的两台设备直接通信，目标mac一定是对方的，但目标ip不一定是对方的，对于路由器接收到目标ip不是自身的话，就转发出去
如： 在A和B两个网络中，两台电脑pc1和pc2要通信,pc1 ip为192.168.1.10  pc2的ip为192.168.2.10
pc1 --> 交换机（可以没有）-->路由器 --> 交换机（可以没有）--> pc2
其中路由器至少有两个网卡，有数据转发功能
发送数据包组装流程：
1.pc1:
    目标ip:192.168.2.10
    目标mac: 路由器网卡1mac
    源ip:192.168.1.10
    源mac:pc1的mac
    默认网关：192.168.1.1（路由器lan网关）
2.数据由交换机转发至路由器，路由器收到后发现数据包不是本网络的
  把数据转发给网卡2，网卡2再组装数据发给网络B
  目标ip:192.168.2.10
  目标mac: pc2mac（变为最终目标的mac）
  源ip:192.168.1.10
  源mac:路由器网卡2的mac
