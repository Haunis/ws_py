1.两台电脑之间的通信
    两台电脑要通信必须<网络号>相同
    网络号 = ip&子网掩码
    ip = 网络号+主机号

    如一台机器,ip 192.168.1.1 子网掩码255.255.255.0
    则网络号:(192.168.1.1)&(255.255.255.0)=192.168.1.0
    主机号:ip-网络号= 0.0.0.1

2.集线器
  简称hub,现在已经不使用了
  多台电脑相连，以广播的形式发信息。其中一台电脑发信息时，其他电脑不可发。工作在物理层
3.交换机
  优化集线器，实现自动寻址，使得一台电脑可以和另外一台电脑通信而不采用广播的形式。
  一台电脑发信息时，其他电脑也可以发。工作在数据链路层
  可以广播也可以单播
    广播:向连接的各个电脑发送arp广播,寻找特定ip的mac地址
    单播:接收到arp广播的电脑,校验信息,发现是自己的ip,将自己的mac地址以单播的形式发给交换机
